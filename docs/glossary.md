# Glossary

- **Autopilot**: Flight controller handling stabilization and low-level flight.
- **Companion computer**: Onboard compute running higher-level autonomy.
- **ConOps**: Concept of Operations, describing how the system is used.
- **Failsafe**: Automated safety behavior on faults (e.g., RTH).
- **Geofence**: Virtual boundary restricting flight area.
- **MAVLink**: Messaging protocol between avionics components.
- **RTH**: Return-to-home behavior.
- **VLOS**: Visual line of sight.
