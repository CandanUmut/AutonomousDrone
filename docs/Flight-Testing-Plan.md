# Flight Testing Plan

## Safety roles
- Pilot-in-command
- Safety officer
- Test director

## Flight test stages
1. **Tether test**: validate control surfaces and throttle.
2. **First flight**: short duration, low altitude, VLOS only.
3. **Stability checks**: level flight and turns.
4. **Return-to-home test**: lost link behavior.
5. **Mission demo**: waypoint mission.

## Go/No-Go gates
- Preflight checklist complete.
- Weather and wind within limits.
- Range safety established.

## Metrics
- Flight duration
- GPS signal quality
- Battery discharge profile
- Control stability

## Logs
- Store logs in `test/flight/reports/` with structured naming.
