# Hardware Workstream

## Responsibilities
- Airframe design and fabrication
- Propulsion and power systems
- Payload integration
- Safety hardware and physical failsafes

## Weekly deliverables
- Updated requirements and BOM
- Prototype build progress
- Bench test results

## Interfaces to software
- Autopilot hardware selection
- Sensor suite specification
- Power and telemetry interfaces

## Acceptance criteria for integration
- Airframe passes static load tests
- Power system validated on bench
- Autopilot and sensors mounted and verified
