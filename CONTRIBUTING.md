# Contributing

Thank you for helping build a safe, open-source autonomous drone platform.

## How to contribute
1. Read **docs/index.md** and **docs/Safety.md**.
2. Open an issue using the templates in `.github/ISSUE_TEMPLATE/`.
3. Create a branch and submit a PR using the PR template.

## Safety-first expectations
- No weaponization or harmful use cases.
- Follow the testing strategy and checklists.
- Log all tests and incidents (see **docs/Testing-Strategy.md** and **docs/Flight-Testing-Plan.md**).

## Code of conduct
By participating, you agree to follow **CODE_OF_CONDUCT.md**.
